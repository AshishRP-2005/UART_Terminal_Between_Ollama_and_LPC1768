# UART_Terminal_Between_Ollama_and_LPC1768
Communication between AI GPT Model Ollama Phi4mini and GSASLPC1768 Microcontroller using UART

## Features
- Input taken using T9 Texting System from keypad on the board
- Input Typed is Displayed on the LCD display of the board
- Input is sent to GPT model usign UART
- Output Generated by AI model is Displayed on LCD Display

## T9 - Texting system
### Concept
- I am using the keypad on the GSAS training kit board
- There are 4x4 keys, set as a matrix, ranging from 0-9, A-F
- The T9 System assigns each key 4 different values
    - Example - key 2 has the values `ABC2`
- Everytime the key is clicked, its value starts with the number and cycles through the associated alpabets based on number of clicks
- Each click must be within 2 seconds of the previous click for the value to change
- If 2 seconds passes then value will be set as the value of the last click

### Implementation
- There is a scanning system, which will keep looping over the 4x4 matrix, waiting for one row to go low and one column to go high, there by crossing that intersection as the currect selected key
- This scanning system will keep running indefinetly

## GLCD - Display
- The onboard glcd display is used to display the input value from the T9 Texting system and Display the output response you get from the laptop through the UART terminal

## UART Terminal
- The UART terminal is used for communication between AI(present in laptop) and microcontroller
- The T9 input from the board is sent through UART to the Laptop
- The python script in the laptop accpets the input, calls the AI model and sends the output response of the AI to the terminal
- The UART recieves the output and displays it on the GLCD display

## Python + LLM 
- The python is script is run through the terminal constantly
- It waits for input from the board through the terminal
- Then calls the LLM and sends question along with master prompt to it
- It sends output prepared by LLM and Refitted by the script itself to the Board through the UART
