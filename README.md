# UART_Terminal_Between_Ollama_and_LPC1768
Communication between AI GPT Model Ollama Phi4mini and GSASLPC1768 Microcontroller using UART

## Features
- Input taken using T9 Texting System from keypad on the board
- Input Typed is Displayed on the LCD display of the board
- Input is sent to GPT model usign UART
- Output Generated by AI model is Displayed on LCD Display
